Aplicação web desenvolvida com foco em apresentar uma **landing page moderna para uma agência criativa**, utilizando recursos visuais avançados em **3D, animações fluidas e vídeos otimizados**, proporcionando uma experiência imersiva sem comprometer a performance.

> ⚠️ Este projeto foi desenvolvido com foco em **estudo, performance e experiência do usuário**, explorando animações avançadas e renderização 3D para aplicações web modernas.

---

## 🚀 Sobre o projeto

A **Agência 3D** é uma landing page interativa criada para simular o site institucional de uma agência digital moderna.

O projeto foi pensado para oferecer:

- Apresentação visual impactante  
- Elemento 3D interativo  
- Animações suaves e modernas  
- Navegação intuitiva  
- Layout totalmente responsivo  

Além disso, toda a estrutura foi desenvolvida com foco em **alta performance e boa experiência do usuário**, especialmente no carregamento de mídias.

## 🎥 Otimização e performance

Um dos principais focos do projeto foi a **otimização dos vídeos e elementos visuais**, evitando lentidão e consumo excessivo de recursos.

Foram aplicadas estratégias como:

- Compressão de vídeos para web  
- Uso de formatos otimizados  
- Carregamento inteligente de mídias  
- Redução de impacto no tempo de carregamento  
- Priorização da experiência em dispositivos móveis  

> O objetivo foi garantir uma experiência fluida, mesmo com recursos visuais pesados como vídeos e objetos 3D.

## 📱 Responsividade

O projeto foi desenvolvido com abordagem **fully responsive**, adaptando-se a diferentes tamanhos de tela:
